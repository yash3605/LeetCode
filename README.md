# LeetCode Solutions

A collection of LeetCode problem solutions organized by algorithmic technique and data structure patterns.

## Directory Structure

- **array&hashing/** - Solutions for array and hash table problems, including topics like duplicates, majority elements, and subarray problems
- **slidingWindow/** - Sliding window technique problems for optimal substring/subarray queries
- **Stack/** - Stack data structure problems and applications
- **twoPointer/** - Two-pointer technique solutions for searching, sorting, and partition problems

## Overview

This repository contains Python implementations of LeetCode problems, organized by the core algorithmic techniques and patterns used to solve them. Each problem is named with its LeetCode number and a descriptive title.

## Language

- Python
